# 📄 AI Resume Analyzer + Job Match System

An intelligent AI-powered system that analyzes resumes and matches them with job descriptions using semantic understanding.

## 🎯 Features

- **AI-Powered Matching**: Uses sentence transformers for semantic understanding
- **Multi-Factor Scoring**: Combines 5 different signals for accurate matching
- **Skills Analysis**: Extracts and categorizes technical and soft skills
- **Skill Gap Analysis**: Identifies matched, missing, and additional skills
- **Visual Dashboards**: Interactive charts and metrics
- **Recommendations**: Personalized suggestions for improvement

## 🚀 How to Use

1. **Upload Resume**: Click "Upload Resume" and select a PDF or paste text
2. **Add Job Description**: Paste the job description in the text area
3. **Analyze**: Click "🚀 Analyze Match" button
4. **View Results**: See comprehensive match analysis with visualizations

## 🛠️ Tech Stack

- Python 3.8+
- Streamlit (UI)
- Sentence Transformers (AI Embeddings)
- spaCy (NLP)
- Plotly (Visualizations)

## 📊 Scoring Methodology

The system uses a weighted scoring approach:
- 40% - Semantic Similarity (AI-based understanding)
- 30% - Skill Match (exact matches)
- 15% - Experience Match
- 10% - Education Match
- 5% - Keyword Density
